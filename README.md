# Tool Man's Tools Backend

This is # Tool Man's Tools app backend. It was built with NodeJs and MongoDB.

## Links

Code in [Github](https://github.com/programming-hero-web-course1/manufacturer-website-server-side-ra89fi)

Live back end in [Heroku](https://secret-plateau-27130.herokuapp.com/)

## Feature

<ul>
  <li>It uses NodeJs as server side technology.</li>
  <li>MongoDB NoSQL database is used as datastore.</li>
  <li>Provides a REST API to front end.</li>
  <li>Uses Express framework.</li>
  <li>It handles CORS.</li>
</ul>
